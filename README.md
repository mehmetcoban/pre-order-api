## Installation

- Clone the repo
- Run `composer install`
- Copy `.env.example` to `.env`
- Run `php artisan key:generate`
- Run `php artisan migrate`
- Run `php artisan db:seed`

## Requirements

- PHP >= 7.3
- MySQL >= 5.7
- Composer
- Laravel Valet (optional)
